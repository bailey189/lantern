About Lantern

Lantern is a lightweight, Raspberry Pi-powered cybersecurity toolkit designed to help small businesses, hobbyists, and professionals illuminate risks in their networks.

Developed with usability and clarity in mind, Lantern combines device discovery, vulnerability detection, and network visualization in a sleek, easy-to-use interface.

Key Features

  • Fast device discovery across Ethernet networks
  • Security scan options using nmap, masscan, arp-scan, and nikto
  • Network visualization using vis.js
  • Device database with editable details
  • Risk-level tagging and web vulnerability tracking

Lantern is licensed and developed by cybersecurity professionals committed to the empowerment of small businesses.
